How to Set Up & Run Your AI Chatbot Online 🚀

Since your chatbot needs a backend to process AI responses, here’s what you need to do:

1️⃣ Set Up the Backend (Server)

We'll use Node.js + Express + OpenAI API to power the chatbot.

Steps:

  1. Install Node.js (Download from here).
  2. Open Terminal/Command Prompt and run:
    bash
    mkdir chatbot-server && cd chatbot-server
  3. Initialize the project:
    bash
    npm init -y
  4. Install required dependencies:
    bash
    npm install express socket.io cors openai dotenv
  5. Create a new file: server.js
  6. Copy & paste this backend code:
javascript
require("dotenv").config();
const express = require("express");
const { Configuration, OpenAIApi } = require("openai");
const http = require("http");
const cors = require("cors");
const { Server } = require("socket.io");

const app = express();
const server = http.createServer(app);
const io = new Server(server, { cors: { origin: "*" } });

app.use(cors());
app.use(express.json());

const openai = new OpenAIApi(
    new Configuration({ apiKey: process.env.OPENAI_API_KEY })
);

io.on("connection", (socket) => {
    console.log("User connected:", socket.id);

    socket.on("message", async (msg) => {
        try {
            const response = await openai.createChatCompletion({
                model: "gpt-4",
                messages: [{ role: "user", content: msg }],
            });

            const botReply = response.data.choices[0].message.content;
            socket.emit("response", botReply);
        } catch (error) {
            console.error("Error:", error);
            socket.emit("response", "Sorry, an error occurred!");
        }
    });

    socket.on("disconnect", () => {
        console.log("User disconnected:", socket.id);
    });
});

server.listen(3000, () => console.log("Server running on port 3000"));

2️⃣ Add OpenAI API Key

  • Create a .env file in your chatbot-server folder.
  • Add this line (replace with your actual OpenAI API key):
